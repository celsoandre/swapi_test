# Star Wars API

## Descrição
Projeto para uso academico, realizando o consumo da api [Star Wars Api](https://swapi.co/), utilizando a linguagem kotlin e os principais frameworks dísponiveis 
no mercado.

## Frameworks :
- [x] [RXJava](https://github.com/ReactiveX/RxJava) versão : 1.2.5

- [x] [RXAndroid](https://github.com/ReactiveX/RxAndroid) versão : 1.2.1

- [x] [Retrofit](https://square.github.io/retrofit/) versão : 2.2.0

- [x] [Adapter RXjava](https://github.com/JakeWharton/retrofit2-rxjava2-adapter) versão : 2.2.0

- [x] [Gson](https://github.com/square/retrofit/tree/master/retrofit-converters/gson) versão : 2.2.0

- [x] [Interceptor](https://github.com/square/okhttp/tree/master/okhttp-logging-interceptor) versão : 3.6.0
